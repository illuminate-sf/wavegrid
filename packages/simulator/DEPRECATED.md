# DEPRECATED

`@wavegrid/simulator` is deprecated and replaced by `@wavegrid/cloud-relay` + `@wavegrid/agent`.

The simulator's scenes/animations have been ported to the QuickJS snippet pattern format and live in the pattern gallery served by `@wavegrid/cloud-relay`. The state engine is now handled by `@wavegrid/agent` running patterns in a sandboxed QuickJS runtime.
