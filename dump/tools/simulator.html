Animation Program Simulator

PC host running animation programs through the ABI. The canvas is rendered as two independent render sets (left + right halves) and stitched — a live demo of the distributed-rendering model. Animations are time-based, so they move at the same speed at any framerate.

—fps
0frame
0pixels driven
Left node — render set rectRuns(0,0,50,100), 2 500 px  ·  Right node — rectRuns(50,0,50,100), 2 500 px.
Each node holds only its own 1-D framebuffer; neither allocates the full canvas. The wave is authored once in global coordinates and is render-set-blind, so the seam is invisible.
